# p1-esamc-17.04.26
Este projeto é o primeiro trabalho avaliativo que possuo na faculdade. Se consiste em um site utilizando HTML, CSS e JavaScript, onde é possível realizar login e senha e um formulário de criação de usuário. Além disso, há uma aba especial para usuários que já possuem seu login e senha.

# Primeira etapa - HTML
Nesse projeto, foi utilizado o HTML, uma linguagem de marcação que funciona como o esqueleto de um site. Sem ele, asplicações web não teriam uma estrutura, um corpo.
